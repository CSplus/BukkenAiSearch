# 不動産物件AI検索システム

## システム概要

ユーザーの希望条件を入力すると、
AIが物件データベースから最適な物件を提案するシステム。

## 開発環境

- Bubble.io
- OpenAI API
- GitHub
- Codex

## 主な機能

### 物件検索

- エリア検索
- 賃料検索
- 坪数検索
- 用途検索

### AI検索

自然言語で検索可能

例：

- 栄駅徒歩5分以内
- 美容室向け
- 居抜き店舗
- 賃料20万円以下

### 管理機能

- 物件登録
- 物件編集
- 物件削除
